# Birthday Websites

A collection of birthday surprise websites built for the Dedalus team. Each site is a custom-built, one-of-a-kind experience designed to ambush the birthday person when they least expect it.

## Projects

### [Shengming](/Shengming)
Rizz-themed birthday surprise overlay injected into the `/pricing` page. When Shengming visits, he gets hit with:
- Full-screen takeover with confetti rain
- Auto-playing rizz music
- Rotating photo gallery with orbiting pictures
- Gen Z captions cycling through ("BRO IS NOT REAL", "THE AURA IS UNMATCHED", etc.)
- Red and gold color palette

### [Cathy](/Cathy)
Cathy's 22nd birthday website.

### [Windsor](/Windsor)
Windsor's birthday website with meme audio and animations.
